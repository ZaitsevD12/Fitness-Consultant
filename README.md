# Golds Gym

Фитнес-консультант: упражнения, диета, специалисты. React-приложение с аутентификацией, персональными планами и файлами.

## Features
- Аутентификация (регистрация/логин).
- Каталог упражнений и блюд (поиск, изображения).
- Специалисты: запись, загрузка медбланков (PDF).
- Персональный профиль: рацион, прогресс, файлы (загрузка/скачивание).
- Подписка: оплата/отмена.
- Админ/спец: управление пользователями, добавление контента.

## Tech Stack
- Frontend: React 18, Redux, Material-UI, Swiper, Axios.
- Backend: Node.js/Express (API на localhost:5005).
- Другое: LocalStorage, FormData для файлов.

## Installation
1. Клонируйте репозиторий: `git clone <repo-url>`.
2. Установите зависимости: `npm install`.
3. Запустите бэкенд: `npm start` (порт 5005).
4. Запустите фронтенд: `npm start` (порт 3000).

## Usage
- Зарегистрируйтесь/войдите.
- Оформите подписку для доступа.
- Просматривайте упражнения/блюда, записывайтесь к специалистам.
- Загружайте файлы в профиле.

## API Endpoints
- `/api/register`, `/api/login`: Аутентификация.
- `/exes/list`: Упражнения.
- `/food/all`: Блюда.
- `/users/pay`: Подписка.
